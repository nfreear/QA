# iet-ou/QA

Rulesets for PHPMD and other quality assurance resources.

Originally for APPLAuD.

## Install ... build ... test

### PHP

```sh
composer install
composer build
composer test
composer execute
composer hook pre-commit
```

### Javascript / Node

```sh
npm i
npm run build
npm test
npm run fix
```
